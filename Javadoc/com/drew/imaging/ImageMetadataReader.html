Class ImageMetadataReader

  • java.lang.Object
    • com.drew.imaging.ImageMetadataReader

  • public class ImageMetadataReader
extends java.lang.Object
    Obtains metadata from all supported file formats, including JPEG, RAW (NEF/CRW/CR2/ORF/RW2), TIFF and PSD.

    If the caller knows their file to be of a particular type, they may prefer to use the dedicated MetadataReader directly (JpegMetadataReader for JPEG files, TiffMetadataReader for TIFF and RAW files, and PsdMetadataReader for Photoshop files).

    The dedicated readers offer a very slight performance improvement, though for most scenarios it is simpler, more convenient and more robust to use this class.
